Adatvédelmi Szabályzat / Privacy Policy

Magyar

A Buresz nevű Discord bot működése során a következő adatokat kezeli:

  • Discord felhasználó azonosító (ID)
  • Felhasználónév
  • Profilkép URL
  • Token alapú játékmenethez kapcsolódó egyenleg (token currency)

Az adatok kizárólag a bot működéséhez szükségesek. A token currency egy **belső játékvaluta**, amelyet például !bwork paranccsal lehet szerezni, valódi pénzt nem igényel, és csak a bot játékszerű funkcióiban használható.

Az adatokat nem osztjuk meg harmadik féllel, és nem kerülnek értékesítésre. A token egyenleg a felhasználó élményének személyre szabására szolgál, különösen a **kaszinó jellegű, de valódi pénz nélküli játékok** esetében.

Az adatkezelés a Discord felhasználási feltételeinek és adatvédelmi szabályzatának megfelelően történik.

English

The Buresz Discord bot processes the following data during operation:

  • Discord user ID
  • Username
  • Profile picture URL
  • Balance of token-based in-bot currency

This data is used solely to ensure proper bot functionality. The token currency is an **in-game, virtual currency** that can be earned via commands like !bwork, and does not require or involve any real money. It is used within the bot’s gameplay features only.

Data is not shared with third parties and is not sold. The balance is stored to personalize the user experience, particularly for **casino-like features that do not involve real-world gambling or money**.

All data usage complies with Discord’s Terms of Service and Privacy Policy.
